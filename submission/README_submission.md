# Submission Package

This folder contains the cleaned deliverables for the panel data replication homework.

## Main deliverables

- `full_replication_answers.md`: main written answers in the order of the template.
- `full_replication_answers.docx`: Word version of the written answers.
- `full_replication_answers.tex`: LaTeX version of the written answers.
- `full_replication_homework.py`: Python code used to generate the analysis and report.
- `full_replication_homework.do`: Stata translation of the main workflow.
- `full_replication_homework.R`: R translation of the main workflow.
- `llm_prompts_and_answers.md`: transcript of the available prompts and answers used in the current completion session.
- `submission_checklist.md`: final checklist against the assignment requirements.

## Data and source material included elsewhere in the project root

- Original paper PDF: `../Household-mobility-and-mortgage-rate-lock_2025_Journal-of-Financial-Economic.pdf`
- Replication package and synthetic data: `../Data for Household Mobility and Mortgage Rate Lock/`
- Assignment template: `../2026 01 17 FTD PANEL Replication Questions Template (2).docx`
- Paper list: `../2026 05 17 FTD PANEL Replication list of papers (2).docx`

## Important methodological note

This is a partial replication using the synthetic public replication package. The original paper's main confidential-data hazard framework is not fully reproduced in Python here. The written report states this clearly.

## PDF note

The local environment allowed reliable generation of DOCX and TEX outputs. A PDF compiler was not installed in the workspace at the time of packaging, so the Word and LaTeX versions are included as the closest final exportable formats from this environment.
