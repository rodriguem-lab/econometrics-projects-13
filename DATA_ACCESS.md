# Data Access

This repository does not include the large replication-package raw data files.

They were excluded for two reasons:

1. GitHub rejected the push because one raw CSV exceeded the 100 MB hard limit.
2. The repository is easier for collaboration when it contains the code, write-up, and compact outputs only.

## Download source

Public replication package:

- https://doi.org/10.17632/74sfv9kx9n.1

## Expected local folder name

After download and extraction, place the folder in the project root as:

- `Data for Household Mobility and Mortgage Rate Lock/`

## What remains in this repository

- the final submission package
- the Python, Stata, and R scripts
- compact tables and figures used for evaluation
- instructions for rebuilding the project
